
# Multiplayer Quiz Show Experience

A modern multiplayer reimagination of the classic *Who Wants to Be a Millionaire* experience, built with Unity and Mirror Networking.

This project transforms the traditional quiz-show format into a competitive, real-time multiplayer experience featuring host-controlled gameplay, audience participation, and high-pressure elimination rounds designed to capture the intensity of a live television game show.

## Features

* Multiplayer quiz gameplay with real-time synchronization
* Host-controlled game sessions
* Competitive 4-player fastest-finger selection round
* 20-question “Hot Seat” challenge system
* Live audience participation support
* Scalable multiplayer architecture
* Interactive game flow and session management

## Tech Stack

* Unity
* Mirror Networking
* C#
* Real-Time Multiplayer Systems

## Technical Highlights

### Real-Time Multiplayer Networking

Implemented low-latency multiplayer architecture using Mirror to ensure synchronized gameplay, smooth player interactions, and stable session management across all connected clients.

### Scalable System Design

Designed the networking architecture to support up to 100 concurrent audience participants while maintaining gameplay responsiveness and performance stability.

### Authentic Quiz Show Mechanics

Developed a structured 20-question progression system inspired by the original game show format, creating tension, competition, and an immersive player experience.

### Host Management Interface

Built a dedicated host control system enabling:

* Session orchestration
* Live player monitoring
* Real-time game state control
* Audience and contestant management
* Match flow administration
