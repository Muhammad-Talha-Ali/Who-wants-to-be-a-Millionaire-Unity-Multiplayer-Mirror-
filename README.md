Project Description
Reimagined the classic Who Wants to Be a Millionaire format into a modern, multiplayer gaming experience. Developed using Unity and Mirror, the project introduces a dynamic Host-controlled environment, a competitive 4-player selection round, and an engaging 20-question “hot seat” challenge designed to replicate the intensity of the original show.

Key Technical Highlights:
• Unity & Mirror Integration: Implemented reliable, low-latency networking to enable seamless real-time multiplayer interactions.
• Scalable Architecture: Designed to support up to 100 concurrent audience participants without compromising performance.
• Game Mechanics: Developed authentic quiz progression with a high-stakes 20-question format, ensuring an immersive and competitive experience.
• Host Management System: Built a comprehensive control interface allowing real-time session monitoring, player management, and gameplay orchestration.

This project demonstrates expertise in multiplayer systems, real-time networking, and interactive game design, delivering a scalable and engaging user experience.
